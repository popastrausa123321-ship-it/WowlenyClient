# 🎮 WowlenyClient - Установка за 3 шага

## ✅ Шаг 1: Скачай файл

Перейди сюда и скачай:
```
https://github.com/popastrausa123321-ship-it/WowlenyClient/releases/download/v1.0.0/wowlenyclient-1.0.0.jar
```

## ✅ Шаг 2: Скопируй в папку mods

**Windows:**
```
C:\Users\[ТвоеИмя]\AppData\Roaming\.minecraft\mods\
```

**Mac:**
```
~/Library/Application Support/minecraft/mods/
```

**Linux:**
```
~/.minecraft/mods/
```

## ✅ Шаг 3: Запусти игру

1. Выбери профиль **Fabric 1.21.4**
2. Запусти
3. Нажми **Right Shift** в игре - откроется меню!

## 🎮 Горячие клавиши

- **Right Shift** - Меню модулей
- **K** - Kill Aura
- **F** - Fly

## ⚠️ Требования

- Fabric 0.16.9+
- Java 21+
- Minecraft 1.21.4

Всё готово! 🚀
